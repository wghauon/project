# 在线教育平台数据库表结构说明

## 概述

本文档描述了在线教育平台的数据库表结构，包含7张核心数据表，涵盖用户管理、课程管理、章节管理、视频管理、课程分类、选课记录和视频学习进度等功能模块。

---

## 表清单

| 序号 | 表名 | 中文名 | 说明 |
|------|------|--------|------|
| 1 | users | 用户表 | 存储学生、教师、管理员三类用户信息 |
| 2 | courses | 课程表 | 存储课程基本信息 |
| 3 | chapters | 章节表 | 存储课程章节信息 |
| 4 | videos | 视频表 | 存储课程视频资源信息 |
| 5 | course_categories | 课程分类表 | 存储课程分类信息 |
| 6 | course_enrollments | 选课记录表 | 存储学生选课记录和学习进度 |
| 7 | video_progress | 视频学习进度表 | 存储学生视频观看进度 |

---

## 1. users 用户表

存储平台所有用户信息，支持学生、教师、管理员三种角色。

### 字段说明

| 字段名 | 数据类型 | 可空 | 默认值 | 说明 |
|--------|----------|------|--------|------|
| user_id | bigint | NO | 自增 | 主键，用户唯一标识 |
| username | varchar(50) | NO | - | 用户名，唯一 |
| password | varchar(255) | NO | - | 加密后的密码 |
| email | varchar(100) | YES | NULL | 邮箱 |
| phone | varchar(20) | YES | NULL | 手机号 |
| real_name | varchar(50) | YES | NULL | 真实姓名 |
| avatar | varchar(255) | YES | NULL | 头像URL |
| role | tinyint | YES | NULL | 角色：1学生/2教师/3管理员 |
| student_no | varchar(20) | YES | NULL | 学号（学生） |
| teacher_no | varchar(20) | YES | NULL | 工号（教师） |
| department | varchar(50) | YES | NULL | 院系 |
| major | varchar(50) | YES | NULL | 专业（学生） |
| status | tinyint | NO | 1 | 状态：0禁用/1启用 |
| created_at | datetime | YES | NULL | 创建时间 |
| updated_at | datetime | YES | NULL | 更新时间 |
| last_login | datetime | YES | NULL | 最后登录时间 |

### 索引

- 主键：`user_id`
- 唯一索引：`username`

---

## 2. courses 课程表

存储课程基本信息，由教师创建，经管理员审核后发布。

### 字段说明

| 字段名 | 数据类型 | 可空 | 默认值 | 说明 |
|--------|----------|------|--------|------|
| course_id | bigint | NO | 自增 | 主键，课程唯一标识 |
| course_name | varchar(100) | NO | - | 课程名称 |
| course_code | varchar(20) | YES | NULL | 课程代码，唯一 |
| teacher_id | bigint | YES | NULL | 外键，授课教师ID |
| category_id | int | YES | NULL | 外键，分类ID |
| description | text | YES | NULL | 课程简介 |
| cover_image | varchar(255) | YES | NULL | 封面图URL |
| difficulty | tinyint | YES | NULL | 难度：1入门/2初级/3中级/4高级 |
| credit | decimal(3,1) | YES | NULL | 学分 |
| hours | int | YES | NULL | 学时 |
| max_students | int | YES | NULL | 最大学生数 |
| semester | varchar(20) | YES | NULL | 学期（如2024春） |
| status | tinyint | YES | NULL | 状态：0草稿/1待审核/2已发布/3已驳回/4已下架 |
| review_remark | varchar(500) | YES | NULL | 审核备注 |
| join_type | tinyint | YES | 1 | 加入方式：1公开/2申请/3邀请码/4关闭 |
| invite_code | varchar(20) | YES | NULL | 邀请码 |
| start_time | datetime | YES | NULL | 开课时间 |
| end_time | datetime | YES | NULL | 结课时间 |
| created_at | datetime | YES | NULL | 创建时间 |
| updated_at | datetime | YES | NULL | 更新时间 |
| published_at | datetime | YES | NULL | 发布时间 |

### 索引

- 主键：`course_id`
- 唯一索引：`course_code`
- 外键：`teacher_id` → users(user_id), `category_id` → course_categories(category_id)

---

## 3. chapters 章节表

存储课程章节信息，用于组织课程内容。

### 字段说明

| 字段名 | 数据类型 | 可空 | 默认值 | 说明 |
|--------|----------|------|--------|------|
| chapter_id | bigint | NO | 自增 | 主键，章节唯一标识 |
| course_id | bigint | YES | NULL | 外键，所属课程ID |
| chapter_no | int | YES | NULL | 章节序号 |
| chapter_name | varchar(100) | NO | - | 章节名称 |
| description | text | YES | NULL | 章节描述 |
| is_required | tinyint | YES | NULL | 是否必修：0否/1是 |
| status | tinyint | YES | NULL | 状态：0禁用/1启用 |
| created_at | datetime | YES | NULL | 创建时间 |

### 索引

- 主键：`chapter_id`
- 外键：`course_id` → courses(course_id)

---

## 4. videos 视频表

存储课程视频资源信息。

### 字段说明

| 字段名 | 数据类型 | 可空 | 默认值 | 说明 |
|--------|----------|------|--------|------|
| video_id | bigint | NO | 自增 | 主键，视频唯一标识 |
| course_id | bigint | YES | NULL | 外键，所属课程ID |
| chapter_id | bigint | YES | NULL | 外键，所属章节ID |
| video_no | int | YES | NULL | 视频序号 |
| video_name | varchar(100) | NO | - | 视频标题 |
| description | text | YES | NULL | 视频描述 |
| video_url | varchar(255) | YES | NULL | 视频文件URL |
| cover_image | varchar(255) | YES | NULL | 视频封面URL |
| duration | int | YES | NULL | 视频时长（秒） |
| file_size | bigint | YES | NULL | 文件大小（字节） |
| format | varchar(10) | YES | NULL | 格式：MP4/AVI/MOV等 |
| resolution | varchar(20) | YES | NULL | 分辨率：如1920x1080 |
| status | tinyint | YES | NULL | 状态：0草稿/1已发布/2已下架 |
| view_count | int | YES | 0 | 播放次数 |
| created_at | datetime | YES | NULL | 创建时间 |
| published_at | datetime | YES | NULL | 发布时间 |

### 索引

- 主键：`video_id`
- 外键：`course_id` → courses(course_id), `chapter_id` → chapters(chapter_id)

---

## 5. course_categories 课程分类表

存储课程分类信息，支持多级分类。

### 字段说明

| 字段名 | 数据类型 | 可空 | 默认值 | 说明 |
|--------|----------|------|--------|------|
| category_id | int | NO | 自增 | 主键，分类唯一标识 |
| category_name | varchar(50) | NO | - | 分类名称 |
| parent_id | int | YES | NULL | 父分类ID，NULL为顶级分类 |
| sort_order | int | YES | NULL | 排序序号 |

### 索引

- 主键：`category_id`

---

## 6. course_enrollments 选课记录表

存储学生选课记录和学习进度。

### 字段说明

| 字段名 | 数据类型 | 可空 | 默认值 | 说明 |
|--------|----------|------|--------|------|
| enrollment_id | bigint | NO | 自增 | 主键，记录唯一标识 |
| course_id | bigint | YES | NULL | 外键，课程ID |
| student_id | bigint | YES | NULL | 外键，学生ID |
| status | tinyint | YES | NULL | 状态：0待审核/1已通过/2已拒绝/3已退课 |
| progress | decimal(5,2) | YES | 0.00 | 学习进度百分比 |
| enrolled_at | datetime | YES | NULL | 选课时间 |
| completed_at | datetime | YES | NULL | 完成时间 |

### 索引

- 主键：`enrollment_id`
- 外键：`course_id` → courses(course_id), `student_id` → users(user_id)

---

## 7. video_progress 视频学习进度表

存储学生视频观看进度。

### 字段说明

| 字段名 | 数据类型 | 可空 | 默认值 | 说明 |
|--------|----------|------|--------|------|
| progress_id | bigint | NO | 自增 | 主键，记录唯一标识 |
| video_id | bigint | YES | NULL | 外键，视频ID |
| student_id | bigint | YES | NULL | 外键，学生ID |
| current_time | int | YES | NULL | 当前观看位置（秒） |
| duration | int | YES | NULL | 视频总时长（秒） |
| progress_percent | decimal(5,2) | YES | NULL | 观看进度百分比 |
| is_completed | tinyint | YES | NULL | 是否完成：0否/1是 |
| watch_count | int | YES | NULL | 观看次数 |
| last_watch_at | datetime | YES | NULL | 最后观看时间 |
| created_at | datetime | YES | NULL | 记录创建时间 |

### 索引

- 主键：`progress_id`
- 外键：`video_id` → videos(video_id), `student_id` → users(user_id)

## 状态码说明

### 用户状态 (users.status)
- 0: 禁用
- 1: 启用

### 用户角色 (users.role)
- 1: 学生
- 2: 教师
- 3: 管理员

### 课程状态 (courses.status)
- 0: 草稿
- 1: 待审核
- 2: 已发布
- 3: 已驳回
- 4: 已下架

### 课程难度 (courses.difficulty)
- 1: 入门
- 2: 初级
- 3: 中级
- 4: 高级

### 视频状态 (videos.status)
- 0: 草稿
- 1: 已发布
- 2: 已下架

### 选课状态 (course_enrollments.status)
- 0: 待审核
- 1: 已通过
- 2: 已拒绝
- 3: 已退课

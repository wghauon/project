## 依赖安装清单

### 后端依赖
```bash
# 密码加密
npm install bcryptjs

# 文件处理
npm install fs-extra

# MD5 计算
npm install spark-md5

# JWT (已安装)
# npm install jsonwebtoken
```

### 前端依赖
```bash
# 视频播放
npm install video.js

# 流程图 (可选)
npm install @vue-flow/core

# 日期处理
npm install dayjs
```
